Detailed Description

Virtual class, to be inherited by all pieces, and to be used as a base class for manipulation on board. It is responsible by storage scope for movements and informations to pass messages to the cells. It uses PAWN type as default for preventing issues and inexpected behavior, but it can be changed to any other type.

Will test the unit piece.h and subclasses to ensure good working of the chess components.
